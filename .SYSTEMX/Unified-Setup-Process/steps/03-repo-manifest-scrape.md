# Step 03 — Repo Manifest Scrape

## Goal
Learn from accessible WayneTechLab repos through structure metadata.

## Actions
- Run the scraper in list/dry-run mode first.
- Run the manifest write only when the operator wants a refreshed snapshot.
- Inspect `.SYSTEMX`, workflows, scripts, docs, Firebase files, and package
  scripts from the manifest.

## Gate
`scrape/repo-manifest.json` exists or the operator intentionally skipped it.
