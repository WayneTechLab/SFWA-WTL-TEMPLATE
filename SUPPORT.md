# Support

Community support is provided through GitHub Discussions and reproducible GitHub
issues. Include the output of `npm run wtl:doctor -- --json`, with sensitive
values removed, plus your operating system and architecture.

The supported lanes are macOS on Apple Silicon and Windows 11 x64/ARM64 with
PowerShell 7. Ubuntu and WSL are experimental. This public template is provided
as-is; use in production is at your own risk.
